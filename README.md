## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| 🇰🇷 **한글 지원** | 한글 ↔ 모스부호 양방향 변환 (SKATS 표준, 자모 분해/조합) |
| 🔄 **양방향 변환** | 텍스트 → 모스부호, 모스부호 → 텍스트 (영어/한글 동시 출력) |
| 🔊 **소리 재생** | Web Audio API로 모스부호 신호음 실시간 재생 |
| 🎚️ **실시간 설정 반영** | 재생 중에도 속도·주파수·볼륨 변경이 즉시 적용 |
| 🔁 **기본값 초기화** | 슬라이더 설정을 한 번에 기본값으로 복원 |
| 📊 **모스부호 차트** | 알파벳 · 숫자 · 특수문자 · 한글 자음 · 한글 모음 (클릭 시 소리 재생) |
| 📖 **모스부호 소개** | 정의, 역사, SOS 신호, 현재 활용까지 한눈에 |
| ⚙️ **작동 원리** | 점/선 길이, 신호·글자·단어 간격을 시각적으로 설명 |
| 📋 **복사 기능** | 변환 결과를 클립보드에 한 번에 복사 |

## 🎮 사용법

### 텍스트 → 모스부호
1. **"텍스트 → 모스부호"** 모드 선택
2. 입력창에 텍스트 입력 — **한글과 영어 모두 가능**
   - 예: `HELLO WORLD`, `안녕하세요`, `Hello 세계`
3. 실시간으로 모스부호가 출력됩니다
4. **▶ 재생** 버튼으로 소리 재생

### 모스부호 → 텍스트
1. **"모스부호 → 텍스트"** 버튼 클릭
2. 모스부호 입력 (예: `.... . .-.. .-.. ---`)
   - 신호 사이: 공백 1개
   - 단어 사이: `/` 사용
3. **EN**(영어)과 **KO**(한국어) 결과가 동시에 표시됩니다

### 사운드 설정
- **재생 속도**: 5~35 WPM (Words Per Minute)
- **주파수**: 300~1000 Hz (톤 높낮이)
- **볼륨**: 0~100%
- **기본값 버튼**: 15 WPM / 600 Hz / 70% 로 즉시 복원
- 재생 중에도 슬라이더를 조절하면 **즉시 반영**됩니다

## 🇰🇷 한글 모스부호 (SKATS)

한글은 [SKATS 표준](https://en.wikipedia.org/wiki/SKATS)에 따라 자모 단위로 변환됩니다.

### 자음
| ㄱ | ㄴ | ㄷ | ㄹ | ㅁ | ㅂ | ㅅ |
|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| `.-..` | `..-.` | `-...` | `...-` | `--` | `.--` | `--.` |

| ㅇ | ㅈ | ㅊ | ㅋ | ㅌ | ㅍ | ㅎ |
|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| `-.-` | `.--.` | `-.-.` | `-..-` | `--..` | `---` | `.---` |

### 모음
| ㅏ | ㅑ | ㅓ | ㅕ | ㅗ | ㅛ | ㅜ | ㅠ | ㅡ | ㅣ |
|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| `.` | `..` | `-.` | `...` | `.-` | `-..` | `....` | `.-.` | `-` | `..-` |

> 쌍자음(ㄲ, ㅆ 등)은 기본 자음을 두 번 전송하고, 복합 모음(ㅐ, ㅔ 등)은 기본 모음으로 분해하여 전송합니다.
